# NzxAntd

`NzxAntd`是一个`angular`组件库，基于`ng-zorro-antd`进行次扩展。全部代码开源并遵循 MIT 协议，任何企业、组织及个人均可免费使用。

## Code scaffolding

Run `ng generate component component-name --project nzx-antd` to generate a new component. You can also use `ng generate directive|pipe|service|class|guard|interface|enum|module --project nzx-antd`.
> Note: Don't forget to add `--project nzx-antd` or else it will be added to the default project in your `angular.json` file. 

## Build

Run `ng build nzx-antd` to build the project. The build artifacts will be stored in the `dist/` directory.

## Publishing

After building your library with `ng build nzx-antd`, go to the dist folder `cd dist/nzx-antd` and run `npm publish`.

## Running unit tests


Run `ng test nzx-antd` to execute the unit tests via [Karma](https://karma-runner.github.io).

## Further help

To get more help on the Angular CLI use `ng help` or go check out the [Angular CLI Overview and Command Reference](https://angular.io/cli) page.
